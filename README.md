# Nudging-ESG-investments
This is the full R script for the paper "Nudging ESG Investments Via Digital Financial Advising. Evidence From an Investment Game Experiment" by Caterina Lucarelli, Matteo Pasquino & Manuele Citi. Forthcoming in "Business Strategy and the Environment" (https://onlinelibrary.wiley.com/doi/10.1002/bse.70582).
